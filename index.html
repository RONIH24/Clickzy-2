"Learning made interactive"

How Does It Help Learning:

  • Increases participation and student engagement
  • Provides immediate feedback for teachers
  • Students can answer anonymously instead of infront of the whole class
  • Provides a break from lectures while maintaining an educational atmosphere
  • Allows students to use familiar and easy to use product

Key Features:

  • Voice recording feature for verbal submissions
  • 4 unique buttons for multiple choice submissions
  • Small compact design for easy storage
  • Ability to use headphone jack for listening and recording assignments
  • Built in microphone for ease of use
